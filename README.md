\# ReinschIt



An AI-powered repair assistant for DIY mechanics. Describe your vehicle problem

and get step-by-step diagnosis and fix instructions through a conversational interface.



\## What It Does

\- Start a repair session by describing your car, truck, or motorcycle problem

\- AI asks follow-up questions and walks you through the fix

\- Save and revisit past repair sessions

\- User accounts to track your repair history



\## Tech Stack

\- Frontend: React + Vite + Tailwind CSS + Shadcn

\- Backend: Go + Gin

\- Database: MySQL

\- AI: Anthropic Claude API

\- Deployment: AWS Lightsail + GitHub Actions



\## Running Locally



\### Frontend

cd frontend

npm install

npm run dev



\### Backend

cd backend

go mod download

go run .
